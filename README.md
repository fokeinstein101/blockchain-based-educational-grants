# Blockchain-Based Educational Grants System

## Overview

A decentralized platform for applying, funding, and tracking educational grants with transparent allocation, automated disbursement, and comprehensive progress monitoring.

## System Description

The Blockchain-Based Educational Grants System revolutionizes educational funding through transparent, automated, and secure grant management on the Stacks blockchain.

## Key Features

### Grant Registry System
- **Grant Creation**: Educational institutions and organizations can create grant programs
- **Eligibility Management**: Automated eligibility verification and requirements tracking  
- **Application Processing**: Streamlined application submission and review processes
- **Transparent Allocation**: Public visibility into grant allocation decisions

### Grant Disbursement System
- **Automated Payments**: Smart contract-based milestone payments
- **Progress Tracking**: Monitor educational progress and grant utilization
- **Compliance Verification**: Ensure funds are used for intended purposes
- **Impact Measurement**: Track educational outcomes and success metrics

## Smart Contracts

### 1. Grant Registry Contract (`grant-registry.clar`)
Lists grants, manages eligibility requirements, and handles application processing.

### 2. Grant Disbursement Contract (`grant-disbursement.clar`)
Releases funds to verified recipients based on milestones and compliance.

## Technical Architecture

- **Blockchain**: Stacks Blockchain with Clarity smart contracts
- **Transparency**: All grant activities recorded immutably
- **Automation**: Smart contract-based milestone payments
- **Verification**: Multi-party verification for large grants
